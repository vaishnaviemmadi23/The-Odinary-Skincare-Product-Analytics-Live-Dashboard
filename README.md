# The Ordinary Skincare Sales Analytics Dashboard

## Overview

This project presents an interactive Sales Analytics Dashboard developed in Microsoft Excel for The Ordinary skincare products. The dashboard transforms raw sales data into actionable business insights through KPI tracking, customer analytics, and data visualization.

The project demonstrates practical data analysis skills including data cleaning, Pivot Tables, Pivot Charts, KPI reporting, customer segmentation, revenue analysis, and dashboard development.

---

## Business Problem

Skincare brands generate large volumes of sales data from multiple products, customer segments, cities, and sales channels. Decision-makers require a centralized dashboard to answer critical business questions:

- Which products generate the highest revenue?
- Which cities contribute the most sales?
- Which purchase channels perform best?
- How do discounts affect revenue?
- Which customer groups are most valuable?
- How satisfied are customers with different products?

This dashboard addresses these questions using Excel-based analytics.

## Key Insights

### Product Performance

- AHA 30% + BHA 2% Peel generated the highest revenue.
- Serum products contributed the highest share of total revenue.

### Geographic Analysis

- Mumbai emerged as the highest revenue generating city.
- Sales were distributed across multiple metropolitan regions.

### Sales Channel Analysis

- Amazon generated the highest revenue among all sales channels.
- Online platforms significantly contributed to total revenue.

### Customer Analysis

- Combination Skin customers generated the highest revenue.
- Customer ratings remained consistently positive across products.

### Revenue Trends

- Monthly analysis revealed fluctuations in sales performance.
- Peak revenue periods were identified through trend analysis.
